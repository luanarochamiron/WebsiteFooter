# Responsive Website Footer - CodingLab

Um rodapé de site (Footer) moderno, completo e totalmente responsivo desenvolvido com HTML5 e CSS3. O projeto apresenta uma estrutura corporativa robusta dividida em colunas de navegação, integração com redes sociais, área de inscrição de newsletter e uma barra inferior de direitos autorais.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica de listas de links, formulário de captura e seções de rodapé.
* CSS3: Estilização com esquemas de cores escuras, tipografia limpa e efeitos de transição.
* Flexbox / CSS Grid: Alinhamento das colunas de conteúdo e distribuição responsiva dos elementos na tela.

## Funcionalidades e Técnicas Aplicadas

* Estrutura Multicolunas: Organização sistemática de links importantes divididos por categorias (Company, Service, Account e Courses).
* Seção de Mídias Sociais: Ícones circulares coloridos e alinhados para as principais redes (Facebook, Twitter, Instagram, LinkedIn e YouTube).
* Formulário de Inscrição (Newsletter): Caixa de entrada de e-mail minimalista acoplada a um botão estilizado "Subscribe".
* Barra de Copyright Separada: Linha inferior discreta contendo os direitos autorais e links para políticas de privacidade e termos de condições.
* Alinhamento Fluido: Uso de propriedades modernas de layout para garantir o espaçamento correto e a adaptabilidade em diferentes resoluções.
